# Orbital Eden

**Hardcore Survival + Industrial Automation + Cozy Elements in Space (Godot 4)**

---

## Overview

Orbital Eden is a multiplayer survival game developed in Godot 4, combining:
- **Extreme survival** (Green Hell style)
- **Industrial automation** (Satisfactory/Factorio style)  
- **Cozy sanctuary building** (Stardew Valley style)

**Core Premise:** Automation as a relief mechanism. The player begins in a hardcore survival situation and transforms their hostile environment into a safe sanctuary through automated systems.

> "Survive the horror, build your paradise."
